## Getting started

Clone.

```bash
git clone git://github.com/resin-io/resin-blog.git
cd resin-blog
```

Install.

```bash
npm install
```

Start your engines.

```bash
npm start
## running production? Add --production
## running on staging server? $ GHOST_CONFIG=staging npm start --production
```

`http://localhost:2368/ghost` :tada:
